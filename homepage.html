Chess-fork game x+y*z: x -- time on the clock when the game starts (in minutes); y -- increment per move from move 1 (in seconds); z -- number of forks available to each player

Aggregate rating game: 1/0 point per board win/loss, 0.5 points per board draw. Player with more points across all boards wins. ELO ratings adjusted once according to this outcome.

Independent rating game: each board is an independent game. Players win, draw or lose in each board. ELO rating adjusted several times (as games finish) according to the outcome in each board.
